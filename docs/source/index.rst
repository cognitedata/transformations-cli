Cognite transformations-cli package
===================================

Command-line interface for Cognite Transformations.


Installation
^^^^^^^^^^^^

To install this package:

.. code-block:: bash

   pip install cognite-transformations-cli

If the Cognite SDK is not already installed, the installation will automatically fetch and install it as well.

To get going, consult the :doc:`quickstart`
